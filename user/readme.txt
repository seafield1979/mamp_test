ユーザー登録システム

ただユーザーを登録するだけのシステム。ログインとホームの表示ができればOK。

■仕様
  ユーザーの情報を管理するシステム。
  初回アクセス時にユーザー登録を行う。次回以降はログイン画面を表示し、ユーザー名＆パスワードを使ってログインする。
  ログイン後のホームでユーザーの情報を参照＆編集できる。
  前回ログインの情報が残っていたらログインなしでホームページに移動。

■ページ構成
  top.html
    未ログイン時のトップページ。ログイン情報のCookieの有無を調べて、Cookieがあったら自動的にログインする。
  login.html
    ログインを行うページ。ユーザー名とパスワードを入力してログイン処理を行う。
    未ログイン時にログインページからログインボタンを押すとこのページに遷移
  home.html
    ログイン後のページ。ユーザー情報を表示。ユーザー情報の編集やユーザー削除、ログアウトができる。
  edit.html
    ユーザー情報を編集するページ。

■データベース
DB:
  user_test
Table:
  Users
    id　ユーザーID
    delete_flag  論理削除
    name ユーザー名
    pass パスワード
    comment ユーザーのコメント
